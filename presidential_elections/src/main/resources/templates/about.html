Project Overview

This project aims to create a transparent and secure platform for the presidential election process. Users can register as voters or candidates, vote for their chosen candidate, and track election results. The goal is to simplify the process of participating in elections and make it more accessible to everyone.

Roles in the Project

This project was developed by a dedicated team of individuals with different expertise:

  • Project Manager: Responsible for overseeing the project, managing deadlines, and ensuring the smooth execution of all phases.
  • Frontend Developer: In charge of designing and implementing the user interface for the platform.
  • Backend Developer: Developed the core functionality, including user management, authentication, and the election logic.
  • UI/UX Designer: Focused on creating a user-friendly and aesthetically pleasing interface for a better user experience.
  • Tester: Responsible for testing the platform and ensuring that all features function correctly before launch.
